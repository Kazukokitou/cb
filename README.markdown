##REQUIRED ENV VALUES

- CB_MAIL_FR:            (Mail-from address)
- CB_MAIL_TO:            (Mail-to address)
- GOOGLE_ANALYTICS_CODE: (Google analytics code)
- BASIC_AUTH_NAME:       (Basic auth name for admin pages)
- BASIC_AUTH_PASS:       (Basic auth password for admin pages)
